# Ноутбуки проекта

- `01_eda.ipynb` - разведочный анализ данных: структура, распределения, спрос по часам, влияние погоды и сезона.
- `02_model_experiments.ipynb` - сравнение моделей по метрикам и просмотр важности признаков.

Боевой код вынесен в `src/`, поэтому ноутбуки служат для анализа и демонстрации экспериментов.
