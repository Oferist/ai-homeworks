# Безопасность и работа с данными

В проекте используются только открытые данные UCI Bike Sharing Dataset. В датасете нет персональных данных пользователей, токенов, паролей или закрытых бизнес-выгрузок.

Правила проекта:

- реальные секреты не хранятся в репозитории;
- `.env` добавлен в `.gitignore`;
- для примера переменных окружения используются `.env.example` и `configs/.env.example`;
- большие или закрытые датасеты не добавляются в проект;
- endpoint `/predict` принимает только числовые признаки и не сохраняет пользовательские запросы.

Если проект будет развиваться дальше, для продакшена стоит добавить rate limiting, централизованное логирование, мониторинг качества модели и контроль доступа к административным endpoints.
