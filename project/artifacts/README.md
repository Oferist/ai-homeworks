# Артефакты проекта

Эта папка содержит результаты обучения и экспериментов:

- `model.joblib` - финальная модель;
- `model_metadata.json` - метаданные обучения;
- `metrics.csv` и `metrics.json` - сравнение моделей;
- `test_predictions.csv` - предсказания на тестовой выборке;
- `feature_importance.csv` - важность признаков финальной модели;
- `*.png` - графики для отчёта и демонстрации.

Модель можно пересоздать командой:

```bash
python -m src.train
```
